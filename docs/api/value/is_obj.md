# **json::Value::is_obj**

```cpp
constexpr bool is_obj() const noexcept;
```

Returns `true` if the inner data is of type [`Object`](../define/Object.md), otherwise returns `false`.

## Return value

Returns `true` if the inner data is a `Object`, otherwise returns `false`.

## Expection

Noexpect.

## Complexity

Constant.

## Version

From v1.0.0 to now.
