# **json::Value::is_nul**

```cpp
constexpr bool is_nul() const noexcept;
```

Returns `true` if the inner data is of type [`Null`](../define/Null.md), otherwise returns `false`.

## Return value

Returns `true` if the inner data is `Null`, otherwise returns `false`.

## Expection

Noexpect.

## Complexity

Constant.

## Version

From v1.0.0 to now.
