# **json::Value::is_str**

```cpp
constexpr bool is_str() const noexcept;
```

Returns `true` if the inner data is of type [`String`](../define/String.md), otherwise returns `false`.

## Return value

Returns `true` if the inner data is a `String`, otherwise returns `false`.

## Expection

Noexpect.

## Complexity

Constant.

## Version

From v1.0.0 to now.
