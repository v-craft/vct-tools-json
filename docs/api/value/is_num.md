# **json::Value::is_num**

```cpp
constexpr bool is_num() const noexcept;
```

Returns `true` if the inner data is of type [`Number`](../define/Number.md), otherwise returns `false`.

## Return value

Returns `true` if the inner data is a `Number`, otherwise returns `false`.

## Expection

Noexpect.

## Complexity

Constant.

## Version

From v1.0.0 to now.
