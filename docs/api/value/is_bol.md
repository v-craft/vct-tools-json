# **json::Value::is_bol**

```cpp
constexpr bool is_bol() const noexcept;
```

Returns `true` if the inner data is of type [`Bool`](../define/Bool.md), otherwise returns `false`.

## Return value

Returns `true` if the inner data is a `Bool`, otherwise returns `false`.

Noexpect.
